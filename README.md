### Research Projects  
- Multi-agent Robotics Research (Python) - University of Waterloo - Designed an exploratory research project using path planning and Multi-Robot Interaction for optimal search.
Data Processing and visualization, including Inferential Statistical Analysis, was conducted using Python.
Publication 1:

Hamid, Owais, Kerstin Dautenhahn, and Chrystopher L. Nehaniv. ”Engineering Social Learning Mechanisms
for Minimalistic Multi-agent Robots.” 2020 3rd International Conference on Control and Robots (ICCR).
IEEE, 2020.  
  

- Virtual Human-Robot Interaction (JavaScript, C#) - University of Waterloo - Designed a study for
virtual Human Robot Interaction study in Unity (Game Engine) utilizing C#. Data processing, Inferential
Statistical Analysis and visualization was performed using R, Python and SPSS. Publication 2: 

Hamid, O.; Chandra, S.; Dautenhahn, K. and Nehaniv, C. (2022). Social Transmission of Information
through Virtual Robotic Agents. In Proceedings of the 14th International Conference on Agents and Artificial
Intelligence - Volume 3, ISBN 978-989-758-547-0, ISSN 2184-433X, pages 361-372  
  

- Evolutionary Robotics (JavaScript) - University of Waterloo Helped design a study that explored
Braitenberg principles, helped automate data collection (JavaScript), and analyzed data using Python (sqlite)
for visualization (seaborn, matplotlib). Publication 3:

Shaghaghi, S., Hamid, O., Nehaniv, C. L. (2021, September). Valentino Braitenberg’s Table: Downhill
Innovation of Vehicles via Darwinian Evolution. In International Conference on Computational Collective
Intelligence (pp. 57-72). Springer, Cham.  
  

- Instrumentation for Hydrogen Evolution Reactions Helped automate instrumentation for data
collection and conducted data analysis using custom software for Hydrogen Evolution Reaction using
Nano-particles. Publication 4:

M Qamar, A Adam, B Merzougui, A Helal, O Abdulhamid, MN Siddiqui, ”Metal–organic framework-guided
growth of Mo 2 C embedded in mesoporous carbon as a high-performance and stable electrocatalyst for the
hydrogen evolution reaction.” Journal of Materials Chemistry A 4.41 (2016): 16225-16232.
